Documentation

This API stores information about various movies for use with the myFlix site.

Endpoint Function URL HTTP Method Body Response
Get a list of all movies /movies GET - A JSON object holding data about all movies
Get data about a single movie by title /movies/[title] GET - A JSON object holding data about a single movie, containing a title, genre, and director.
Get data about a genre by name /movies/genre/[name] GET - A JSON object holding a description of a genre
Get data about a director by name /movies/director/[name] GET - A JSON object holding data about a director containing name, date of birth, and date of death
Add a new user /users POST A JSON object holding the user's name, structured like: {name: "kevin"} A JSON object holding the user's name that was added in addition to a blank array for favorite movies and an ID: {"name": "kevin", "favoriteMovies": [], "id": "12"}
Change user's name /users/[id] PUT A JSON object holding the new name for the user: {"name": "Richard Updated Name"} A JSON object holding the data for the user with the new user name: {"name": "Richard", "favoriteMovies": [], "id": "12"}
Add a movie to user's favorite movies /user/[id]/[title] POST - Text stating that the movie was successfully added
Remove a movie from user's favorite movies /user/[id]/[title] DELETE - Text stating the movie was successfully removed
Delete a user's account /user/[id] DELETE - Text stating the user was successfully deleted
