Documentation

This API stores information about various movies for use with the myFlix site.

Endpoint Function URL HTTP Method Body Response
Get a list of all movies /movies GET - A JSON object holding data about all movies.
{
'Title': 'movie title',
'Description': 'movie description',
'Genre': {
'Description': 'genre description'
},
'Director': {
'Name': 'director name',
'Bio': 'director bio',
'Birthday': 'director birthday'
}
'ImageUrl': 'imageurl',
'Featured': true or false
}
Get data about a single movie by title /movies/[title] GET - A JSON object holding data about a single movie.
Get data about a genre by name /movies/genre/[name] GET - A JSON object holding a description of a genre.
Genre: {
Description: 'genre description'
},
Get data about a director by name /movies/director/[name] GET - A JSON object holding data about a director.
'Director': {
'Name': 'director name',
'Bio': 'director bio',
'Birthday': 'director birthday'
}
Add a new user /users POST A JSON object holding the user's username, structured like: {'Username': 'kevin647'} A JSON object holding the user's name that was added in addition to a blank array for favorite movies and an ID.
{
'name': 'kevin647',
'favoriteMovies': [],
'id': '12'
}
Change user's name /users/[id] PUT A JSON object holding the new name for the user.
{'Username': 'Richard Updated Username'} 		A JSON object holding the data for the user with the new user name.
{
'username': 'Richard',
'favoriteMovies': [],
'id': '12'
}
Add a movie to user's favorite movies /user/[id]/[title] POST - Text stating that the movie was successfully added
Remove a movie from user's favorite movies /user/[id]/[title] DELETE - Text stating the movie was successfully removed
Delete a user's account /user/[id] DELETE - Text stating the user was successfully deleted
